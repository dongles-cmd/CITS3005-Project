Project Resources

Explore the key tools, libraries, and datasets that power the iFixit Knowledge Graph Project. Below you'll find tutorials, documentation, and videos to help you understand and utilize these resources efficiently.

Libraries & Frameworks

Owlready2

Owlready2 is a Python library that helps in creating and manipulating OWL ontologies, allowing reasoning, querying, and ontology modifications.

RDFLib

RDFLib is a powerful library for handling RDF data. It enables creating a knowledge graph, managing triples, and running SPARQL queries.

Flask

Flask is a lightweight web framework used to build the web interface for interacting with the knowledge graph and ontology.

Data Sources

iFixit Dataset

The iFixit dataset contains repair guides, tools, and parts information for a wide range of devices sourced from the iFixit community.

Custom Ontology

A custom ontology built from the iFixit dataset represents the structure of repair guides, tools, steps, and parts.
