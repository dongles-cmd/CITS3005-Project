Project Resources

This project makes use of several tools, libraries, and datasets to build and manage the iFixit Knowledge Graph. Below is an overview of the key resources used:

Libraries & Frameworks

  • Owlready2

    Owlready2 is used for creating and manipulating OWL ontologies. It enables reasoning, querying, and modifying ontologies using Python. This library powers the ontology representation and validation in the project.

  • RDFLib

    RDFLib is a Python library used for working with RDF (Resource Description Framework) data. It allows us to create a knowledge graph, manage triples, and execute SPARQL queries over the graph.

  • Flask

    Flask is a lightweight web framework for Python. It's used to create the web interface that allows users to interact with the knowledge graph and ontology via a browser.

  • FontAwesome

    FontAwesome provides the icons used throughout the web interface, improving the visual layout and user experience.

Data Sources

  • iFixit Dataset

    The iFixit dataset is a collection of repair guides and manuals sourced from the iFixit community. It includes detailed steps, tools, and parts information for repairing various devices.

  • Custom Ontology

    A custom ontology is built from the iFixit dataset to represent procedures, items, parts, tools, and steps. This ontology is created using OWLready2 and is key to understanding the structure of repair guides in a semantic way.

Additional Resources & Tutorials

  • Owlready2 Ontology Guide

    A tutorial on creating and managing OWL ontologies using Owlready2. This guide was fundamental in structuring the ontology used in this project.

  • RDFLib Documentation

    This documentation covers how to create and query RDF graphs using RDFLib, which was used to implement the knowledge graph and handle SPARQL queries.

  • Flask Web Framework Tutorial

    A comprehensive tutorial on building web applications using Flask. This resource helped in building the interface for interacting with the ontology and knowledge graph.

  • Google Charts

    Although optional, Google Charts can be used to visualize the data in the knowledge graph. This might be used for future visualization enhancements.
